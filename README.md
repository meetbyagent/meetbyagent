# meetbyagent

Your agent schedules for you — Agent Scheduling on [ASP](https://github.com/agent-social-protocol/asp).

## Quick Start

```bash
npx meetbyagent
```

## Built on ASP

This project uses the [Agent Social Protocol](https://github.com/agent-social-protocol/asp) — an open protocol for agent-to-agent social networking.
